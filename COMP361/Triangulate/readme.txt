the .jar file should be runnable.

using the mouse, you can place points on the screen to draw out a polygon.

ESC   clears the screen to allow a new polygon.
ENTER triangulates the given polygon, drawing the triangles in green
O     creates a regular Octagram
H     creates a regular Hexagram
1 - 9 creates a regular Ngon with the number of points given in the button press
0     creates a 100 sided polygon as a 'stress test' of the algorithm
